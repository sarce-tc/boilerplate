# /arquitectura — Router de micro-skills

Carga SOLO las skills que necesita tu tarea. No leas todo — elige por fila.

| Tarea | Skills a cargar |
|---|---|
| Agregar command handler | `/arq-stack` · `/arq-command` · `/arq-repos` |
| Agregar query handler | `/arq-stack` · `/arq-query` · `/arq-repos` |
| Modificar lógica de dominio | `/arq-domain` · `/arq-command` |
| Agregar endpoint en controller | `/arq-controller` · `/arq-errors` |
| Escribir / modificar repositorio Dapper | `/arq-schema` · `/arq-repos` |
| Entender / depurar error handling | `/arq-errors` |
| Duda sobre convenciones o librería | `/arq-rules` |
| Aggregate nuevo end-to-end | `/arq-stack` · `/arq-schema` · `/arq-domain` · `/arq-repos` · `/arq-command` · `/arq-query` · `/arq-controller` · `/arq-new` |
| Code review de un handler | `/arq-command` o `/arq-query` · `/arq-rules` |
| Orientación general del proyecto | `/arq-stack` · `/arq-rules` |
